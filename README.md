## Problem
Predict customers at risk of churn to support retention strategies and business decision-making.

## Approach
- Data preprocessing and feature engineering
- Feature selection
- Ensemble learning

## Models
- XGBoost
- Random Forest
- LightGBM

## Results
- Improved churn prediction performance using ensemble models
- Focused on identifying high-risk customers
